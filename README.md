# BoneLOG

**File-based Blazor WASM blog engine**

- **[Live site](https://taqiam.github.io/BoneLog/)**
- **[Documentation](https://taqiam.github.io/BoneLog/posts/en/00001/index)**

## Features
- Markdown-based
- Multilingual
- Tags
- Categories
- Search & Filter


> [!NOTE]
> This project is no longer actively maintained.
>
> It started as a university project and was later expanded into a more complete blog engine.
> While it remains functional and can still be used or extended, I do not plan to continue its development because I no longer consider Blazor WebAssembly the right technology choice for this type of application.
