| What | File |
|------|------|
| Theme, layout, markdown | `wwwroot/css/app.css` |
| Code highlighting | `wwwroot/css/prism-tomorrow-night.css` |
| Dark mode | Header toggle → `dark` class on `<body>` |
| RTL text | Auto on Arabic/Persian paragraphs |

Edit source files in the repo, not the published `gh-pages` output. Redeploy after CSS changes.

Use CSS variables in `app.css` for colors (light/dark).
