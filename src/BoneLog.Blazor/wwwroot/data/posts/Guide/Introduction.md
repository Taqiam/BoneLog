**BoneLog** is a file-based blog built with **Blazor WebAssembly**. You write Markdown, push to Git, and deploy a static site. No database, no CMS, only files under `wwwroot/data` plus a generated `index.json` for the home page.


## What you get

- Home page with post list, search, and optional sidebars (categories, tags, languages)
- Post pages at `/post/{path}`
- About page and custom Markdown pages
- Dark mode, syntax highlighting, Mermaid diagrams
- GitHub Actions for index generation and deployment
